# Key:mind-numbing
## Response

E-1: The lecture on calculus was so <u>mind-numbing</u> that I almost fell asleep.
E-2: Watching political debates can be <u>mind-numbing</u> because they often repeat the same points over and over.
E-3: Data entry can be <u>mind-numbing</u> work, but it needs to be done accurately.
