# Key:on the same page
## Response
### English Sentences
E-1: We need to have a meeting to make sure we're all <u>on the same page</u> about the project.
E-2: I thought we were <u>on the same page</u>, but it seems like we have different priorities.
E-3: Let's make sure we're <u>on the same page</u> before we present this idea to the boss.
